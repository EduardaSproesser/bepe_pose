# Data Availability — `midia/`

This folder centralizes media and supporting material for the FAPESP project (grant **2025/05613-8**).

## How this folder is organized

- `cad_files/`
	- Contains CAD files and mechanical design assets used in the experimental setup.

- `presentations/`
	- Contains slides and presentation files related to project milestones and results.

- `images/`
	- Contains photos and image records of the setup, experiments, and project documentation.

- `DATA_AVAILABILITY.md`
	- This document, describing where media content is located and how to access it.
